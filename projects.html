Client Sentiment Analysis Framework

A modular NLP system built to quantify mood shifts in financial news and online discussions. I analyzed cross-platform sentiment trends across Bloomberg, Reuters, Reddit, and X to understand how macroeconomic events influence public tone and investor anxiety.

  • Processed 50,000+ text samples using custom token filters and domain-tuned rules
  • Achieved 94 percent parsing accuracy after iterating on negation and stopword handling
  • Identified sentiment swings tied to CPI releases, unemployment reports, and Fed meetings
  • Built 15+ comparative visualizations showing divergence between institutional and retail sentiment
  • Designed a reusable pipeline structure ready for future ML classification modeling

MBTA Ridership Analysis (2018–2025)

A multi-layered exploration of Boston’s transit system using Python, Matplotlib, Altair, and Vega-Lite. I analyzed systemwide recovery patterns, event-driven demand, and line-level variation while building interactive dashboards for user exploration.

  • Analyzed 6+ years of ridership across 120+ stations
  • Identified pandemic-phase ridership collapse and plateau at ~70–80 percent recovery
  • Found that Red Sox home games increased Kenmore Station entries by 20–45 percent
  • Built interactive dashboards for line-level and station-level patterns
  • Created weather-driven demand visualizations using Vega-Lite
